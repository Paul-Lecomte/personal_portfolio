# Paul Lecomte — Personal Portfolio

This is a minimal React + Vite + Tailwind CSS portfolio site for showcasing backend and systems projects.

## Getting started

```powershell
cd C:\Users\lecom\OneDrive\Images\personal_projects\personal_portfolio
npm install
npm run dev
```

Then open the printed localhost URL in your browser.

## Available scripts

- `npm run dev` — start the development server.
- `npm run build` — build the production bundle.
- `npm run preview` — preview the production build locally.

## Customisation

- Update contact details in `src/components/Contact.tsx`.
- Adjust project content in `src/data/projects.ts`.
- Tweak layout and styling in the components under `src/components`.
